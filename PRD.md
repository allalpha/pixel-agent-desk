  ┌──────────┬───────────────────────────────┬──────────────────┬────────────────┐
  │   상태   │          Hook 이벤트          │     캐릭터 동작   │     라벨       │
  ├──────────┼───────────────────────────────┼──────────────────┼────────────────┤
  │ Waiting  │ SessionStart, SessionEnd, Idle │ 가만히 서 있기   │ Waiting...     │
  ├──────────┼───────────────────────────────┼──────────────────┼────────────────┤
  │ Working  │ UserPromptSubmit, PreToolUse,  │ 바쁘게 움직임    │ Working...     │
  │          │ PostToolUse, SubagentStart/Stop│ (8fps)           │ (+경과 시간)   │
  ├──────────┼───────────────────────────────┼──────────────────┼────────────────┤
  │ Done     │ Stop, TaskCompleted           │ 인사, 작업 완료   │ Done!          │
  │          │                               │ (6fps)           │ (+최종 시간)   │
  ├──────────┼───────────────────────────────┼──────────────────┼────────────────┤
  │ Error    │ PostToolUseFailure            │ 느낌표 (4fps)    │ Error!         │
  ├──────────┼───────────────────────────────┼──────────────────┼────────────────┤
  │ Alert    │ Notification, PermissionRequest│ 느낌표 (4fps)    │ Help!          │

  ---
  기술 스택

  Electron (Node.js + HTML/JS/CSS)
  ├── HTTP Local Server (동적 포트 할당, 기본 3456)
  ├── IPC 통신
  ├── CSS Animation (스프라이트 시트)
  └── Claude CLI Hooks (type: http)

  ---
  아키텍처 흐름

  Claude CLI Hooks
       ↓ (HTTP POST)
  Electron Main Process (HTTP Server + IPC)
       ↓ (IPC Renderer)
  Electron Renderer Process (애니메이션 + 상태 라벨)

  ---
  📁 파일 구조

  ```
  pixel-agent-desk/
  ├── main.js           # Electron 메인 프로세스, HTTP 서버, 훅 관리

  ├── styles.css        # 디자인 시스템 (Glassmorphism, 픽셀 렌더링, drag)
  ├── package.json      # 의존성 관리
  └── avatar_00.png     # 픽셀 캐릭터 스프라이트 시트 (48x64, 9x4 프레임)
  ```

  ---
  🔧 실행 방법

  ```bash
  # 1. 의존성 설치
  npm install

  # 2. 앱 실행
  npm start
  # 또는
  npx electron .
  ```

  Claude CLI에서 hooks가 자동으로 `~/.claude/settings.json`에 등록됩니다.

  ---
  📊 HTTP API

  - `POST /agent/status` - 상태 업데이트 수신
    ```json
    {
      "session_id": "string",
      "hook_event_name": "string",
      "message": "string"
    }
    ```

  - `GET /agent/states` - 현재 상태 조회
  - `GET /health` - 서버 상태 확인
